# BiblioApp — Gestion de Bibliothèque

Mohamed ADMANE — M1 MIAGE — Avril 2026

## Description
Cette application web permet de gérer le catalogue d’une bibliothèque municipale. Elle offre la possibilité de consulter, ajouter, modifier et supprimer des livres, ainsi que de rechercher un ouvrage par titre ou auteur. L’application inclut aussi une navigation simple et un formulaire avec validation des données. L’API sera simulée avec json-server, sans développement backend.


## Prérequis
- Node.js v18+
- npm install -g @angular/cli
- npm install -g json-server

## Lancer le projet
- npm install  
- Terminal 1 : json-server --watch db.json --port 3000  
- Terminal 2 : ng serve  
- Ouvrir : http://localhost:4200

## Fonctionnalités réalisées
- Ajouter une barre de navigation
- Consulter la liste des livres disponibles
- Consulter, Ajouter, modifier et supprimer un livre (CRUD complet)
- Rechercher un livre par titre ou par auteur
- Naviguer entre les différentes sections de l'application
- Remplir un formulaire d'ajout avec validation des données
- Utiliser un API Json
